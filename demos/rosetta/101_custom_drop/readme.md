# 101 - Custom Drop

## 目标特性 (Target Feature)
展示 Rust 的自定义 `Drop` trait 是如何被编译器前端降级的。

## 降级逻辑预演 (Expected Lowering Logic)
1. **显式调用**：高级语言中的“隐式释放”在 SA-ASM 中被彻底抹除。前端必须在变量作用域结束（或早期返回）之前，显式调用对应的 `@drop_xxx(&self)` 析构函数。
2. **所有权释放**：在调用完析构函数后，前端依然需要发射 `!reg` 来物理释放寄存器本身占用的内存或追踪标记。
3. **递归析构**：对于嵌套结构体，前端必须展平嵌套的 Drop 调用图，按正确的顺序（从内到外或从外到内）依次发射 `call` 和 `!` 指令。
